# hack_g_a_1

Pequeño proyecto de ejemplo en Python que ejecuta un script simple y muestra un mensaje en consola.

## Descripción

Este repositorio contiene un solo archivo Python (`app.py`) que imprime un mensaje de prueba en la terminal.

## Contenido

- `app.py`: Script principal que ejecuta la aplicación.

## Requisitos

- Python 3.x

## Uso

1. Abre una terminal en la carpeta del proyecto.
2. Ejecuta:

```bash
python app.py
```

3. Deberías ver en la salida:

```bash
Hola, esto es una prueba
Hola, esto es una prueba
```

## Notas

Esta aplicación es un ejemplo mínimo de ejecución de un script Python.
